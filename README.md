# AI 輔助網絡統合分析教學

Network Meta-analysis with R - 醫學統計教學課程

## 課程內容

透過 **prompt engineering** 學會網絡統合分析（Network Meta-analysis）的實作課程。

### 任務列表

1. 理解網絡統合分析的基本概念
2. 網絡幾何（Network Geometry）
3. 執行網絡統合分析與解讀結果
4. 評估一致性（Consistency）
5. 評估異質性（Heterogeneity）
6. 排名分析（Ranking）
7. 次群體分析與網絡統合迴歸
8. 發表偏誤評估（Publication Bias）
9. 敏感度分析（Sensitivity Analysis）
10. 完整學術報告撰寫（PRISMA-NMA）

```r

```

```bash

```

### 渲染

